## 一、机制修复

### 1. 须佐能乎 — 流血触发时机调整

我的文字描述是每一段增加一段流血；目前的效果是打完之后才增加流血；这样少了很多流血的伤害。举个例子，本来是0段流血，造成 7 * 6 的伤害，如果按照现在的机制是42点伤害打完之后才给6个流血；而我希望的效果是：

```
7 -> (1流血) -> 7+1 -> (2流血) -> 7+2 -> (3流血) -> 7+3 -> (4流血) -> 7+4 -> (5流血) -> 7+5 -> (6流血)
```

请你修改流血的追加！**只是修改流血追加的增加时机；本身的数值、升级后的效果都保留不变。**
